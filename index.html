Deep learning based abnornmal event detection in padestrian pathways

[Github Link]

Developed a Deep Learning-based Abnormal Event Detection System for pedestrian pathways using the UCSD anomaly detection dataset. The system processes video frames through preprocessing steps like resizing, grayscale conversion, and background subtraction. A hybrid CNN-LSTM model was implemented to extract spatial and temporal features, enabling detection of unusual activities such as running, loitering, or sudden crowd movement. The model evaluates anomalies using reconstruction error, temporal coherence, and optical flow analysis. Performance was measured with metrics like AUC-ROC, Precision, Recall, and F1-Score. The project also includes a visualization dashboard to flag events with timestamps and anomaly scores, simulating real-world deployment for surveillance in public spaces.
